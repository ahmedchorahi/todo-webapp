# ToDo App

A sleek, responsive, and dynamic ToDo web application built with HTML, CSS, and vanilla JavaScript.

## ✨ Features
- **Add, edit, and delete** your daily tasks.
- **Mark tasks as completed** (completed tasks are pushed to the bottom of the list).
- **Persistent Storage**: Saves your tasks locally in the browser so they persist across sessions (using localStorage).
- **Modern UI**: A dynamic design with an eye-catching neon accent color (`#00ffc4`), custom checkboxes, and smooth hover animations.
- **Responsive Layout**: Designed to look great on both desktop and mobile devices.

## 🛠️ Technologies Used
- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Media Queries, SVG Icons)
- **JavaScript** (ES6+, DOM Manipulation, Local Storage)

## 🚀 How to Run Locally
1. Clone the repository or download the source code.
2. Open the `index.html` file in any modern web browser.
3. Start managing your tasks!
